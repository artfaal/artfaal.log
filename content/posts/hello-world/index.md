Это первый пост в моём новом блоге **artfaal.log**. Здесь я буду публиковать длинные материалы, которые не помещаются в формат Telegram Instant View.

## Как создавать новые посты?

### Структура Page Bundle

Каждый пост создается как отдельная директория:

```
content/posts/my-new-post/
├── index.md          # Основной файл поста
├── cover.jpg         # Обложка поста (опционально)
├── image1.jpg        # Изображения для поста
└── image2.png        # Любые другие файлы
```

### Front Matter

В начале `index.md` укажите метаданные:

```yaml
---
title: "Заголовок поста"
date: 2026-01-01T12:00:00+03:00
draft: false
tags: ["тег1", "тег2"]
author: "artfaal"
description: "Краткое описание поста"
cover:
    image: "cover.jpg"
    alt: "Описание обложки"
    caption: "Подпись к обложке"
    relative: true
---
```

### Использование изображений

Просто положите изображение в папку с постом и используйте:

```markdown
![Описание](image.jpg)
```
